# **JOGO DO NUMERO SECRETO**

## *SOBRE*

Trata-se de um jogo simples criado no curso de Lógica de Programação oferecido pela Alura.  
É um jogo de tentativas para adivinhar o numero sorteado, enquanto aparecem dicas na tela.
